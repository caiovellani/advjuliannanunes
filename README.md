# advjuliannanunes

## About the Project

This project was developed for lawyer Julianna Nunes' website, providing a professional and modern platform to showcase legal services.

## Technologies Used

* Next.js
* React
* TailwindCSS
* ShadCN
* Vercel
* pnpm

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/caiovellani/advjuliannanunes.git
   ```
2. Install the dependencies:

   ```bash
   pnpm install
   ```
3. Start the development server:

   ```bash
   pnpm dev
   ```

## Deployment

The project is hosted on Vercel.
